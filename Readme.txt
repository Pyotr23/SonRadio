Команды в последовательном порте для управления радио:
    - 00xxx(x) - установка частоты, где ххх(х) - значение частоты в кГц ("Шоколад" - 00980).
    Значение частоты в кГц записывается в 0 и 1 байты EEPROM. В 0 байт - старший байт значения частоты.
    - 01x(x) - установка громкости, где x(x) - значение громкости, ограниченное диапазоном 0 - 18.
    - 02x - управление подачей входного сигнала.
